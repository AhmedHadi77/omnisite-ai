# Case Study Draft

## Problem

Agencies often manage client websites across Webflow, WordPress, and Shopify with separate tools, disconnected tasks, and slow review loops.

## Solution

OmniSite AI creates one operating dashboard for website health, content operations, AI recommendations, and client approvals.

## What I would build next

- Real OAuth for each platform.
- Background sync jobs with Redis queues.
- Approval-based publishing workflows.
- Agency analytics and reporting exports.
